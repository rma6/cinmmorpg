# Projeto de IP


Comunicação entre jogos vai ser dada através do socket.

Compilando e rodando:
# Compilando o server:
	$ make server 
# Compiando o client:
	$ make client
# Compilando client e server:
	$ make
# Rodando server:
	$ ./server
# Rodando client:
	$ ./client
	
O nome dos arquivos podem ser modificados no arquivo makefile
